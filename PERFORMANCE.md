## 使用方法

### 開発環境でのパフォーマンス監視
```bash
npm run dev
```
開発サーバー起動時に自動的にパフォーマンス監視が開始されます。

### 本番ビルドの最適化
```bash
npm run build
```
最適化されたバンドルが生成されます。

### パフォーマンステスト
```bash
npm run preview
```
本番ビルドをローカルでプレビューできます。
